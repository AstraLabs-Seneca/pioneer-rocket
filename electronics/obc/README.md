# OBC — On-Board Computer

**Lead:** Sai

## Scope
- STM32F405RGT6 setup (clock, reset, USB-C)
- Flight sensors: IMU, altimeter, GPS + antenna
- High-speed data storage: Dual SD (SDIP + SPI), dual Flash
- LoRa 433MHz telemetry
- CAN transceiver bus + protection
- Recovery system ejection logic (apogee detection)
- KiCad PCB schematic, design rules, layout

## Tools
- **KiCad** — PCB design
- **STM32CubeIDE** — Firmware development
